About Obadh Engine

Obadh Engine is a linguistic Bengali transliteration engine that converts Roman text to Bengali. It uses an algorithmic approach based on Bengali phonology.

Features

  • Linguistically accurate transliteration
  • Support for all Bengali writing complexities
  • Conjuncts (juktakkhor) handled algorithmically
  • Proper rendering of reph (র্), hasanta (্), and pholas
  • Fast and efficient with minimal memory usage

How It Works

Obadh Engine uses a 4-step transliteration process:

  1. Tokenization: Breaks input text into meaningful linguistic units
  2. Phonological Analysis: Converts tokens to Bengali phonemes
  3. Syllable Formation: Organizes phonemes into syllables
  4. Orthographic Rendering: Formats syllables into proper Bengali text

The engine is built in Rust and compiled to WebAssembly for web usage.
